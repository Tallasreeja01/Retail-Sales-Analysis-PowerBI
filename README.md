# 📊 Retail Sales Analysis Dashboard

## 📌 Project Overview

The **Retail Sales Analysis Dashboard** is an interactive Power BI dashboard developed to analyze retail sales performance and provide meaningful business insights. The project demonstrates the complete workflow of a Data Analyst, starting from data cleaning and transformation to dashboard creation and business reporting.

The dashboard enables users to monitor key business metrics, compare sales performance across different dimensions, identify trends, and interactively filter the data for deeper analysis.

---

# 🎯 Project Objectives

- Analyze overall retail sales performance.
- Track important business KPIs.
- Compare sales across regions, categories, and customer segments.
- Identify sales trends over time.
- Create an interactive dashboard for business decision-making.

---

# 🛠️ Tools & Technologies Used

- **Microsoft Excel** – Initial data exploration
- **SQL** – Data querying and analysis
- **Power Query** – Data cleaning and transformation
- **Microsoft Power BI** – Dashboard development and visualization

# 📂 Dataset

**Dataset Used:** Sample Superstore Dataset

The dataset contains retail order information, including:

- Order ID
- Order Date
- Ship Date
- Region
- Category
- Sub-Category
- Segment
- Sales
- Profit
- Quantity
- Discount

---

# 🧹 Data Cleaning & Preparation

The dataset was cleaned using **Power Query** by performing the following steps:

- Imported CSV dataset
- Verified column names
- Corrected data types
- Converted date columns
- Checked for missing values
- Removed duplicate records
- Trimmed extra spaces
- Cleaned text values
- Verified categorical consistency
- Performed basic anomaly detection
- Identified data outliers
- Loaded the cleaned dataset into Power BI

---

# 📈 Key Performance Indicators (KPIs)

The dashboard tracks the following KPIs:

- 💰 Total Sales
- 💹 Total Profit
- 📦 Total Quantity Sold
- 🧾 Total Orders

# 🔍 Business Insights

- Which product category performs the best?
- Which customer segment contributes the highest profit?
- How have sales changed over time?
- How do sales and profit vary across regions and categories?
